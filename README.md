# 电压型逆变器双闭环仿真

## 资源描述

本资源文件提供了三相电压型逆变器的SIMULINK仿真模型，采用了双闭环控制策略。该仿真模型通过将电压和电流解耦变换到旋转dq0坐标系中进行控制，实现了对逆变器的精确控制。

## 仿真细节

- **控制方法**：双闭环控制，电压和电流解耦变换到旋转dq0坐标系。
- **开关管电流峰值**：约45A。
- **负载线电压有效值**：220V。
- **频率**：50Hz。
- **负载电压波形**：三相对称。
